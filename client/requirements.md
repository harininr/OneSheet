## Packages
framer-motion | For smooth page transitions and UI animations
clsx | For conditional class merging
tailwind-merge | For merging tailwind classes safely
lucide-react | For beautiful icons
react-dropzone | For drag and drop file uploads
html2canvas | For downloading the cheat sheet as an image
jspdf | For downloading as PDF

## Notes
- Upload endpoint is POST /api/upload (multipart/form-data)
- Process endpoint is POST /api/cheatsheets/:id/process
- Frontend needs to poll or wait for processing to complete
- Design uses strict Sage Green (#9CAF88) color palette
